Costco Prototype Using Java

Description: A prototype for an online Costco store built with Java and integrated with Spring Security for user authentication and authorization. Users can sign up, log in, and access restricted sections based on roles and permissions.

Date: March 2025

Technologies: Java, Spring Boot, Spring Security, Thymeleaf, H2 Database

Features:

  • User registration and login with authentication using Spring Security
  • Authorization to restrict access to certain parts of the site based on user roles (e.g., admin, customer)
  • Integration of a shopping cart and product catalog
  • Responsive design with Thymeleaf templates for UI rendering

Challenges: Ensuring proper session management, handling user roles, and securing sensitive data such as user credentials.
